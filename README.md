## form-receiver

> A simple form submission receiver in express, TypeScript, and Nodemailer

## Installing

1. Clone repo 
  ```bash
  git clone https://github.com/ijjk/form-receiver
  ```

2. Install dependencies
  ```bash
  yarn install
  ```

3. Build
  ```bash
  yarn build
  ```

## Testing

1. First build if you haven't already
  ```bash
  yarn build
  ```

2. Run tests with AVA
  ```bash
  yarn test
  ```
